This page demonstrates how to include source code with syntax highlighting using the Prism.js library. Note the use of the language-javascript, language-python, and language-java classes.

Example: Java Hello World

The **Java Hello World** program is the traditional first step in learning Java. It defines a public class named HelloWorld and includes the necessary public static void main(String[] args) method, which serves as the entry point for execution by the Java Virtual Machine (JVM). The actual output is produced using System.out.println("Hello, World!").


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Example: JavaScript Function

Here is a small function to calculate the Fibonacci sequence:


function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

// Call the function
console.log(fibonacci(10));

Example: Python Script

Python code highlighting also works automatically:


# A simple Python script
def greet(name):
    """Prints a greeting message."""
    print(f"Hello, {name}!")

if __name__ == "__main__":
    greet("DevDocs User")
