Interesting Problems

Problem Approach
Set Matrix Zeroes Use a matrix cell instead of a new variable to save space.
Implement Queue using Stacks Use two stacks.
Daily Temperatures Same as next greater element. Use stack.
Remove k adjacent duplicates Use stack. Keep a pair of char and count in the stack. See design gurus.
Remove k digits Use stack. Many edge cases: Leading zeros, replaced < k, empty result to "0". See design gurus.
Longest Palindrome Create a frequency map. No need to sort. s.length() - oddFreqCharsCount + 1 will also work.
Permutation in String How to compare two permuations? Use freq map. Use array instead of hashmap. Saves time in hash calculation, unboxing objects. If using map, use intValue() as in map.get(c).intValue()
k-Nearest Duplicates Find duplicates within a distance of k in an array. Use set to keep only the k nearest elements. Look it up.
Count Binary Substrings 001110000 -> 0011, 01, 111000, 1100, 10. Answer = 5. Count zeros, ones and call it a group array. group = {2,3,4}. Sum of Math.min(i-1, i) combinations can be made.
Third maximum number Use Long.MIN_VALUE instead of Integer.MIN_VALUE if that is a valid input.
Arranging Coins For large numbers, beware of doing n * (n + 1). Don't have few ints and few doubles. Have all variables as double.
Reverse an Integer Why reversing an int need not be an int? The Integer.MAX_VALUE and Integer.MIN_VALUE are 2147483647 and -2147483648 respectively. Consider 1534236469. Reversing it goes off bounds of Integer.
Intersection of Two Arrays One nice intersection strategy. Iterate through array. if array element ispresent in set, we found an intersecting element. remove that element from set to avoid future re-addition.
