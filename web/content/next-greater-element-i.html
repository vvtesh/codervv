Example: Next Greater Element I Solution (Monotonic Stack)

This code solves the "Next Greater Element I" problem by efficiently finding the first element greater than a given element in another array. It utilizes a **Monotonic Stack** to process the larger array, nums2, in a single pass ($O(N)$ time complexity) while maintaining a decreasing order of elements. A **HashMap** stores the mappings from an element to its next greater element, allowing the final array, res, to be constructed quickly by iterating through nums1 and retrieving the pre-computed results.
