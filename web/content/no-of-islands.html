Example: Number of Islands Solution (DFS)

This Java code solves the "Number of Islands" problem, which asks to find the total number of islands in a 2D binary grid. The approach uses Depth-First Search (DFS) to identify and mark entire islands. When the main loop encounters '1' (land), it increments the island count and calls removeIsland, which recursively explores all adjacent land cells (up, down, left, right) and changes them to '0' (water). This process effectively "sinks" the entire island, preventing it from being counted again, until the entire grid is traversed.
