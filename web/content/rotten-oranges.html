h3>Example: Rotting Oranges Solution (BFS)

This code solves the "Rotting Oranges" problem by finding the minimum time required for all fresh oranges to become rotten. The approach uses a multi-source Breadth-First Search (BFS), where all initially rotten oranges are added to the queue at time zero. The BFS processes contamination in discrete, minute-long rounds, using a special (-1, -1) marker in the queue to track the passage of time. Finally, if any fresh oranges remain after the process, it means they were unreachable, and the function returns $-1$, otherwise it returns the total time elapsed.
