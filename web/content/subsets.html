Given an integer array nums of unique elements, return all possible subsets (the power set).

    Example Input/Output
    Input: nums = [1,2,3]
    Output: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

Approach 1: Simple Backtracking

Uses a binary decision tree. At each step, we pick or don't pick a number.

Approach 2: Looped Backtracking

We use a n-ary decision tree where n = len(nums).




Think about it. We add to results a copy as we process.

Process(0):
    Add [].
    Add 1 to list.
        Process(1)
    Remove 1.
    Add 2 to list.
        Process(2)
    Remove 2.
    Add 3 to list.
        Process(3)
    Remove 3.

Output:
    Iter1
      Iter2
        Iter3

    Output of print statement. Adding indentation for easier understanding.
    []
    [1]
      [1, 2]
        [1, 2, 3]
      [1, 3]
    [2]
      [2, 3]
    [3]

Approach 3: Bit mask trick

This complication can be reduced by going from 2^n to 2^(n+1) so that binary number has fixed width with a 1 as MSB. So, we will get a fixed length binary string.
